Bootstrap Basics

There are two options for using Bootstrap.

  1. Download the files and use bootstrap.css as you would any other .css file, or
  2. Link to the online version at https://stackpath.bootstrapcdn.com/bootstrap/4.4.1/css/bootstrap.min.css.
